## 序列的+和*操作

---
```
lists = list(range(4))
result1 = lists * 2
result2 = lists + lists
print(result1, result2)

[0, 1, 2, 3, 0, 1, 2, 3]
```
---
* #### 对其进行```+```和```*```操作,实际上并不改变原有的操作对象,只是构造一个全新的序列
-[x] 注意,这里如果要在列表里构造新的列表,不能依靠上述的方法,因为它们会指向同一的列表
```
false_list = [[''] * 3] * 3
false_list[1][1] = '4'
print(false_list)

[['', '4', ''], ['', '4', ''], ['', '4', '']]
```
### 由于它们只是复制了列表,但是都是指向同一个列表,所以赋值会覆盖所有列表的相应位置数值
```
true_list = [[''] * 3 for i in range(3)]
true_list[1][1] = '4'
print(true_list)

[['', '', ''], ['', '4', ''], ['', '', '']]
这里仅仅改变了第一行第一列的数值,符合要求
```
