## 切片的操作

* ### 一维切片
>#### 切片的赋值
```
import numpy
l = list(range(10))
print(l)
```

>#### 删除切片
```
del l[1:3]
print(l)
```
>#### 对切片进行赋值 ,注意这里赋值的对象是必须是一个可以迭代的对象,否则会报错
```
l[1:3] = [100, 200]
print(l)
```
* ## 多维切片
-[x] 由于py中仅支持一维切片的索引,所以可以利用外部库来解决多维问题
```
numpy.ndarray()
```
#### 就可以用这种形式来```a[i,j]```来获取,即调用```a.__getitem__```和```a.__setitem__```方法来进行读取

* ## ```省略号...```
#### 它可以当作切片的一种规范形式,也是numpy中对多维的一种快捷方式
```
f(a, ..., z)
```
